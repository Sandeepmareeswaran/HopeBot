# HopeBot 🤖

A mental wellness mobile application designed to provide emotional support
and guidance through an AI-powered chatbot interface.

## 📱 About
HopeBot is a compassionate AI chatbot mobile app built to offer users
a safe space to express their feelings, get mental health tips, and
access supportive conversations anytime.

## 🛠️ Tech Stack
- **Frontend:** React Native (Expo)
- **AI/Chatbot:** LLM API Integration
- **Backend:** Node.js / Firebase
- **Database:** Firebase Firestore

## ✨ Features
- AI-powered conversational chatbot
- Mood tracking and journaling
- Mental wellness tips and resources
- Clean and calming UI/UX

## 🚀 Getting Started

### Prerequisites
- Node.js >= 18
- Expo CLI
- Android/iOS Emulator or physical device

### Installation
```bash
git clone https://github.com/Sandeepmareeswaran/HopeBot.git
cd HopeBot
npm install
npx expo start
```
